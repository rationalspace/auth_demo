# This project has 2 parts - front-end and back-end
## Front-end :
### Angular (http://localhost:4200)
### Components: 
    Register ( uses template form )
    Login ( uses reactive form )
    Admin
### Routes:
    /login
    /register
    /admin
    /home

## Back-end: 
### Express+Node (http://localhost:3000/)
### Module dependencies:
    bryptjs
    cors
    bodyparser
    sqlite3
